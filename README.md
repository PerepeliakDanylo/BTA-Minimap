# BTA-Minimap (Ukrainian / Українська)

Minimap Mod for Better Than Adventure!

This is a fork of [MartinSVK12/BTA-Minimap](https://github.com/MartinSVK12/BTA-Minimap) with the UI fully translated into Ukrainian.

## Installation

1. Download `minimap-1.6.0.jar` from [Releases](https://github.com/PerepeliakDanylo/BTA-Minimap/releases)
2. Place it in your BTA `mods/` folder (replacing the original minimap jar if present)

## Changes

- All GUI elements translated to Ukrainian (menus, settings, waypoints, buttons)
- No gameplay changes  only UI text
